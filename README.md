# RCYC — Weekly Meta Ads Deck

This repository contains the weekly Meta Ads performance deck and its accompanying update playbook for RCYC.

## Files

| File | Description |
|------|-------------|
| `RCYC_Weekly_Meta_Ads_Deck_Update_Playbook.pdf` | Step-by-step playbook for updating the weekly Meta Ads performance deck, including date range definitions, campaign list, conversion metric rules, slide-by-slide update instructions, and the weekly checklist. |
| `Q1_2026_Meta_Paid_Social_Performance_Review.pptx` | The weekly Meta Paid Social performance review deck (Q1 2026). |

## Playbook Summary

The playbook covers:
- **Date ranges:** Last 28 Days, MTD, and 90-Day Benchmark windows
- **Campaigns:** RAQ (6 campaigns) and RAB (4 campaigns)
- **Conversion metric rules:** `cost_per_result` only — never leads
- **Slides updated automatically:** RAQ Global, RAQ US/CA, MTD RAQ, MTD RAB, RAB Global, RAB US/CA
- **Slides requiring manual input:** Top Creative (Slides 7 & 8), Country slides
- **Weekly checklist** to ensure consistent, accurate updates each week
